# Summary
<p>This repository contains the main code used for model training and evaluation that is reported in the manuscript
<i>Improving cardiopulmonary exercise testing guided preoperative risk prediction in non-cardiac surgery by adressing body-specific and sex-related differences through a machine learning model. A secondary post hoc analysis of the METS study.</i></p>
<p>This manuscript is currently under review in <i>The Lancet Digital Health</i></p>

## Data Protection
Patient data originates from the METS study and is not publicly shared outside the METS study group.

## Codebook
>`main.R` - R script for data preprocessing and model training and model evaluation.
>`Functions/` - Folder containing custom functions called via `main.R`
